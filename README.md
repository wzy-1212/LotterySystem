这是一个基于WPF的抽奖系统



双击LotterySystem.exe运行（如果不能运行，可以打开LotterySystem源码->LotterySystem->bin->Release->net8.0-windows->LotterySystem.exe）



进入界面后，点击下方的**开始抽奖**即可开始抽奖：

点击一次抽取一等奖，抽取结束后可以**继续抽奖**，抽取二等奖，依次。。。



所有奖项抽取完毕后会出现”所有人已抽完“的弹窗，点击下方的**重置抽奖**即可重新开始抽奖



左下角有两个按钮：编辑奖项和导入参与人员



**编辑奖项**：

点击编辑奖项，可以在奖项列表里选中需要编辑的奖项，然后下方的名称和人数两个输入框会自动填上，并且可以进行更改，将需要更改的名称和人数修改之后，点击更新奖项即可更新



如需添加奖项，在编辑奖项页面的名称和人数里输入好需要添加的奖项，之后点击添加奖项，即可依次添加在奖项列表下方



如需删除奖项，需要在奖项列表里选中需要删除的奖项，然后点击删除奖项即可删除



在编辑奖项页面点击确定之后会自动重置抽奖



**导入参与人员**：

导入参与人员需要导入excel文件，excel文件需满足前三列位工号，姓名和部门，如本文件夹里的”参与抽奖人员.xlsx“所示，第一行为标题行，不会进行导入。

可以点击导入参与人员按钮，然后双击选中本文件夹里的”参与抽奖人员.xlsx“进行导入（”参与抽奖人员.xlsx“为事先准备好的已经填写过的excel表格）



窗口右侧有一个**主题**按钮，点击可以变换按键的颜色。
